小狼毫输入法——五笔86 小仙并击版
==================================

## 开发说明
感谢乳糖可耐同学告知小狼毫可写并击输入。

参考[佛振](mailto:chen.sst@gmail.com)的[宮保拼音](https://code.google.com/p/rimeime/wiki/ComboPinyin)，对小狼毫自带的五笔86码表进行修改，添加键位为[ejoson](mailto:ejoson@126.com)的[小仙并击](http://xbeta.info/xiaoxian.htm)键位。
